Si usamos texto, es mejor colocar el enlace dentro de un párrafo para modificarlo:

¡Este enlace te lleva a Google!

O también dentro del enlace:

¡Este enlace te lleva a Google!
El atributo target nos permite manejar cómo se abre nuestro enlace, si le asignamos el valor target="_blank" el enlace será abierto en una pestaña nueva.
