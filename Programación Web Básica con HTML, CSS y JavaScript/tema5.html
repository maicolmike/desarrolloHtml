Programación Web

La Programación Web es una disciplina que se ha popularizado mucho en los últimos años. El crecimiento del internet ha permitido la creación de millones de páginas web nuevas, que ofrecen funcionalidades y servicios a una gran cantidad de usuarios.

Se utilizan 3 principales lenguajes para ella, los cuales son HTML, CSS y JavaScript. Cada uno cumple una función específica.

HTML

Este lenguaje le da estructura a las páginas.

CSS

Este lenguaje es para decorar páginas.

JavaScript

Este lenguaje es para manipular elementos de las páginas.
